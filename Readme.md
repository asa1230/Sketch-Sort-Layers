### 图层整理

整理选中的图层或画板.提示:
插件会重新排列顺序，为了不影响排列后的视觉效果。在使用前，请把完成的元素转化为组。

不过，插件也能返回重做图层的顺序或重新设置画板的位置

**选项顺序:** `Text (A->Z)`, `Text (Z->A)`, `Layer Name (A->Z)`, `Layer Name (Z->A)`, `Top`, `Left`, `Random`

**快捷键:** `ctrl` + `⌘` + `⌥` + `S`
